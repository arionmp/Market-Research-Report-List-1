The main types include Ultrasonic Testing, which uses high-frequency sound waves to detect flaws; Radiography Testing, which uses X-rays or gamma rays for inspection; Visual Inspection Testing, which relies on visual examination; Magnetic Particle Testing and Electromagnetic Testing, which detect surface and subsurface flaws using magnetic fields; Eddy-Current Testing, which evaluates electrical conductivity; Liquid Penetrant Testing, which detects surface cracks; and Others, such as infrared testing, acoustic emission, terahertz imaging, and thermography testing.</p></p>
